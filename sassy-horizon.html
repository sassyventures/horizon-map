Horizon Roadmap — first time setup
01
Create a Supabase project
Go to supabase.com, create a free project. Name it anything, e.g. horizon-roadmap.
02
Run this SQL in the SQL Editor
In Supabase go to SQL Editor and run this. Click to copy:
Click to copy
create table initiatives (
  id uuid primary key default gen_random_uuid(),
  sort_order integer default 0,
  sprint text default 'now',
  cat text default '',
  area text default '',
  title text default '',
  priority text default 'Medium',
  size text default 'M',
  value text default '$$',
  problem text default '',
  why text default '',
  scope text default '',
  kpi text default '',
  notes text default '',
  ref text default '',
  jobs jsonb default '[]',
  created_at timestamptz default now()
);
alter table initiatives enable row level security;
create policy "public access" on initiatives
  for all using (true);
03
Paste your credentials below
