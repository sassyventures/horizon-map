Horizon Roadmap: First time setup
1
Create a free Supabase project
Go to supabase.com, sign up free, and create a new project. Name it anything, e.g. keep-roadmap.
2
Create the initiatives table
In Supabase go to SQL Editor and run this. Click to copy:
Click to copy
create table initiatives (
  id uuid primary key default gen_random_uuid(),
  sort_order integer default 0,
  sprint text default 'now',
  cat text default '',
  area text default '',
  title text default '',
  priority text default 'Medium',
  size text default 'M',
  value text default '$$',
  problem text default '',
  why text default '',
  scope text default '',
  kpi text default '',
  notes text default '',
  ref text default '',
  jobs jsonb default '[]',
  created_at timestamptz default now()
);
alter table initiatives enable row level security;
create policy "public access" on initiatives
  for all using (true);
3
Paste your project credentials below
Find these in Supabase under Project Settings → API.
Your credentials are saved in your browser's localStorage and never sent anywhere except directly to your Supabase project.
